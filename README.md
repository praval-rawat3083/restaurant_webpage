# Restaurant Landing Page 🍽️

A modern and responsive restaurant landing page built using HTML and CSS. The website showcases food items, services, and menu sections with an attractive user interface.

## Features

- Responsive design
- Hero section with promotional offer
- Food menu showcase
- Service highlights
  - Discount Voucher
  - Fresh Healthy Food
  - Fast Home Delivery
- Clean and modern UI
- Smooth layout using Flexbox/Grid

## Technologies Used

- HTML5
- CSS3
